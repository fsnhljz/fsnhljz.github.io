Mac微信客户端WeChatTweak防撤回安装教程

2022-06-03 #微信

    • 功能
    • 安装教程

功能

消息防撤回
WechatTweak 可以阻止微信消息撤回，对于好友已经撤回的消息，你依然可以看到。
微信多开
安装 Tweak 后，在程序坞的微信图标上单击右键，就可以登录另一个微信，支持无限多开。
免二次认证登录
安装 Tweak 后，打开微信的设置，在 Tweak 的设置中，开启「免认证登录」，这样就只有在第一次登录 Mac 微信时，需要手机扫码验证登录，如果中间退出了微信，再登录时可直接登录，无需手机二次确认。
右键复制链接
还支持识别二维码、导出表情、Alfred workflow、Launchbar action

安装教程

官网下载
下载WeChatTweak-CLI

双击解压下载的「WeChatTweak-macOS-master.zip」
将下载的「wechattweak-cli」放入上面解压的文件夹中
打开「终端」App （在启动台-其他文件夹中）
在「终端」App 中输入 cd ,并敲一个空格，然后把解压的文件夹拖到「终端」 App 中（此步骤是为了获取 WeChatTweak 的路径）
在终端中粘贴这段命令并回车执行

xattr -d com.apple.quarantine wechattweak-cli

再将下面的命令粘贴到终端，回车执行

chmod +x wechattweak-cli

继续下面的命令粘贴到终端，回车执行，提示输入开机密码，在 password 后面输入开机密码，输入的时候不会显示任何字符，直接输，输完敲回车（这一步的目的是执行安装）

sudo ./wechattweak-cli --install

然后会发现无法打开「微信」
我们需要重启 Mac ，然后登录并打开微信的设置，有 Tweak ，则大功告成。

如何卸载
终端输入下面命令并回车

sudo ./wechattweak-cli --uninstall
